# Word Count Task

## Objective
Read a text file from an absolute path, count the number of words in the file,
and write the result to an output file.

## Input
- The input file is located at: `/app/input.txt`
- The file contains plain text with words separated by whitespace.

## Processing
- Read the contents of `/app/input.txt`
- Count the total number of words in the file
- A word is defined as any sequence of characters separated by whitespace

## Output
- Write the word count to `/app/output.txt`
- The output file must contain a single integer representing the total word count

## Constraints
- Use absolute paths only
- Do not hardcode the output
- The output must depend on the contents of the input file
